Pickle Rick - TryHackMe

Intro

Welcome to my first ever CTF writeup. I created this both to develop my skills and to help beginners learning Web Application Penetration Testing.

This Rick and Morty themed challenge requires us to exploit a web server and locate three ingredients to turn Rick back into a human.

From the introduction we know:

  • We have a target machine IP
  • We are attacking a web server
  • Three flags must be found

Flag One

First we perform enumeration using Nmap:

nmap -A -v 10.130.170.42

The -A flag performs aggressive scanning including service and OS detection.

The scan reveals two open ports:

  • 22 - SSH
  • 80 - HTTP

Inspecting the source code reveals a hidden username:

Username: R1ckRul3s

We search for hidden directories using Gobuster:


gobuster dir -u http://10.130.170.42 \
-w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt \
-x php,html,txt

Important directories discovered:


/index.html
/login.php
/robots.txt

Flag Three

Checking the root directory:

We find:

Copying it into the web directory:
