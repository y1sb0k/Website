TryHackMe Room Writeup: Web Application Penetration Testing

TryHackMe Room

🔗 Link to Room: Pickle Rick - THM

Intro:

Welcome to my first ever CTF Writeup! I am writing this writeup to both help develop myself and my skills, whilst hopefully guiding others who are new to the field of Web Application Penetration Testing. Please forgive any unusual styling or writing choices I make, as I develop my skillset! Without further ado, lets dive in the Pickle Rick room by TryHackMe! (For reference, I have already completed this THM Room, and I will be re-creating and documenting my steps.)

In this room, we are given very little context, only:

This Rick and Morty-themed challenge requires you to exploit a web server and find three ingredients to help Rick make his potion and transform himself back into a human from a pickle.

From this small intro, we know three things:

  1. The target IP - 10.130.170.42 -Note that this will be different for you for your own machine.
  2. We are attacking a web server
  3. There are 3 flags to find.

Flag One:

Let's start with some initial enumeration. Using Nmap will allow us to take a look at the server, specifically identifying ports and services it may be running that may reveal potential vulnerabilities. I have used the command:

nmap -A -v 10.130.170.42

As shown in the output, we have two open ports, ssh running on port 22, and http running on port 80. Navigate to http://10.130.170.42:80.

Check the source code for hidden clues. Right there in the HTML source code, we have a Username commented out.

Username: R1ckRul3s

Use Gobuster to search for hidden directories:

gobuster dir -u http://10.130.170.42/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,html,txt

Checking robots.txt reveals "Wubbalubbadubdub". Testing this on the login page works!

Inside the Command Panel, ls reveals Sup3rS3cretPickl3Ingred.txt. Since cat is blocked, we view it via URL.

Flag 1: mr. meeseek hair

Flag Two:

Find the second ingredient by checking /home/rick using ls -la.

To read the file with a space in the name:

while read line; do echo $line; done < /home/rick/second\ ingredients

Flag 2: 1 jerry tear

Flag Three:

Check the root directory using sudo: sudo ls -la /root. Copy the flag to the web directory to view it.

sudo cp /root/3rd.txt ./3rdFlag.txt

Flag 3: fleeb juice

Summary:

1. mr. meeseek hair
2. 1 jerry tear
3. fleeb juice
