Pickle Rick - THM Writeup

  • Back to TryHackMe Writeups

Intro

Welcome to my first ever CTF Writeup! I am writing this writeup to both help develop myself and my skills, whilst hopefully guiding others who are new to the field of Web Application Penetration Testing. Please forgive any unusual styling or writing choices I make, as I develop my skillset! Without further ado, lets dive in the Pickle Rick room by TryHackMe!

(For reference, I have already completed this THM Room, and I will be re-creating and documenting my steps.)

In this room, we are given very little context, only:

  • The target IP - 10.130.170.42 *-Note that this will be different for you for your own machine.*
  • We are attacking a web server
  • There are 3 flags to find

Flag One

Let's start with some initial enumeration.

Using Nmap will allow us to take a look at the server, specifically identifying ports and services it may be running that may reveal potential vulnerabilities. I have used the command:

nmap -A -v 10.130.170.42

The -A flag is an Aggressive Scan (includes -sV, -O, -sC, traceroute). -v is verbose.

As shown, we have two open ports: SSH on 22, HTTP on 80. Navigating to http://10.130.170.42:80:

Viewing the page source reveals a commented username:


<!--
Note to self, remember username!
Username: R1ckRul3s
-->

Next, using Gobuster to find hidden directories:

gobuster dir -u http://10.130.170.42/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,html,txt

We find /index.html, /login.php, /robots.txt. Viewing robots.txt:

Using username R1ckRul3s and password Wubbalubbadubdub at /login.php:

Denied page appears for non-admin users:

Command panel enumeration using ls:

File Sup3rS3cretPickl3Ingred.txt contains first flag. Accessing via URL IDOR:

Flag 1: mr. meeseek hair

Flag Two

Using /clue.txt and checking /home/rick:

Second flag file is /home/rick/second ingredients, space escaped with backslash:

while read line; do echo $line; done < /home/rick/second\ ingredients

Flag 2: 1 jerry tear

Flag Three

After some further enumeration and checking /root:

Third flag file copied locally:

sudo cp /root/3rd.txt ./3rdFlag.txt

Flag 3: fleeb juice

Summary

All flags collected:


1. mr. meeseek hair
2. 1 jerry tear
3. fleeb juice

We have effectively obtained the ingredients list to turn Rick back into a human!
