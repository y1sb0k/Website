Intro

Welcome to my first ever CTF Writeup! I am writing this writeup to both help develop myself and my skills, whilst hopefully guiding others who are new to the field of Web Application Penetration Testing. Please forgive any unusual styling or writing choices I make, as I develop my skillset! Without further ado, let's dive in the Pickle Rick room by TryHackMe!

In this room, we are given very little context, only:

This Rick and Morty-themed challenge requires you to exploit a web server and find three ingredients to help Rick make his potion and transform himself back into a human from a pickle.

From this small intro, we know three things:

  1. The target IP - 10.130.170.42 (Note: will differ for your machine)
  2. We are attacking a web server
  3. There are 3 flags to find

Flag One

Let's start with some initial enumeration using Nmap:

nmap -A -v 10.130.170.42

The -A flag performs aggressive scan including service/version detection, OS detection, default scripts, and traceroute. -v is verbose mode.

Two ports are open: SSH on 22 and HTTP on 80. Navigating to the website shows:

Inspecting the source code reveals a username:

Username: R1ckRul3s

Using Gobuster to enumerate hidden directories:

gobuster dir -u http://10.130.170.42/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,html,txt

Interesting results in robots.txt:

Login page attempt:

Username: R1ckRul3s
Password: Wubbalubbadubdub
