Pickle Rick – TryHackMe Writeup

This Rick and Morty themed challenge involves exploiting a vulnerable web server to locate three ingredients needed to turn Rick back into a human.

Enumeration

First we perform an Nmap scan.

nmap -A -v 10.130.170.42

The scan reveals two open ports:

  • 22 - SSH
  • 80 - HTTP

Web Enumeration

Opening the webpage hosted on port 80:

Inspecting the page source reveals a hidden username.

Username: R1ckRul3s

Next we perform directory enumeration using Gobuster.

gobuster dir -u http://10.130.170.42 -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,html,txt

We discover a robots.txt file.

Wubbalubbadubdub

Login

Using the discovered credentials we log in.

Inside the admin panel we find a command execution interface.

Listing files reveals the first ingredient file.

Final Ingredient

Listing the root directory.
