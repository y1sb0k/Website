Pickle Rick

TryHackMe Writeup

  • Back to TryHackMe Writeups

Intro

Welcome to my first ever CTF Writeup! I am writing this writeup to both help develop myself and my skills, whilst hopefully guiding others who are new to the field of Web Application Penetration Testing.

Please forgive any unusual styling or writing choices I make, as I develop my skillset! Without further ado, lets dive in the Pickle Rick room by TryHackMe!

(For reference, I have already completed this THM Room, and I will be re‑creating and documenting my steps.)

This Rick and Morty-themed challenge requires you to exploit a web server and find three ingredients to help Rick make his potion and transform himself back into a human from a pickle.

From this small intro we know three things:

  • The target IP
  • We are attacking a web server
  • There are 3 flags to find

Flag One

Let's start with some initial enumeration.

Using Nmap will allow us to take a look at the server, specifically identifying ports and services it may be running that may reveal potential vulnerabilities. 

nmap -A -v 10.130.170.42

We can see two open ports: SSH (22) and HTTP (80).

Checking the page source reveals a hidden username.

Username: R1ckRul3s

Directory Enumeration

gobuster dir -u http://TARGET_IP -w directory-list-2.3-medium.txt -x php,html,txt

Interesting directories discovered:

  • /login.php
  • /robots.txt

The robots.txt file contains a suspicious string: 

Wubbalubbadubdub

This works as the password for the login page.

Command Injection

After logging in, we gain access to a command panel.

Running ls reveals a file:

Sup3rS3cretPickl3Ingred.txt

Cat is blocked, but the file can be accessed directly via URL.

Flag 1

mr. meeseek hair

Flag Two

Flag 2

1 jerry tear

Flag Three

Flag 3

fleeb juice

Summary


1. mr. meeseek hair
2. 1 jerry tear
3. fleeb juice

We have effectively obtained the ingredients list to turn Rick back into a human!
